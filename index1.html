Personal Trainer Dashboard
  1. Go to console.cloud.google.com → Create a new project.
  2. Enable Google Sheets API under APIs & Services → Library.
  3. Go to APIs & Services → OAuth consent screen. Set type to External, fill in app name, add your email as a test user.
  4. Go to Credentials → Create Credentials → OAuth 2.0 Client ID. Choose Web application.
  5. Under Authorised JavaScript origins, add the URL where you'll host this file (e.g. https://yourname.github.io or http://localhost:8080).
  6. Copy the Client ID and paste it above.
  7. Create a blank Google Sheet. Copy the ID from the URL — it's the long string between /d/ and /edit.
  8. Paste the Sheet ID above and sign in. The app will auto-create all tabs and populate a standard workout library on first run.
⚡ Hosting tip: Drop this file into a GitHub repo and enable GitHub Pages for a free permanent URL. No server required.
